##### 技术栈

前端 [Next.js](https://nextjs.org) / [Nuxt.js](https://nuxt.com) / 或者后端模板引擎渲染

管理后台 [Vite](https://vitejs.dev) + [React](https://react.dev) + [Antd](https://ant-design.antgroup.com/index-cn)

服务端 Golang / Rust / Node 提供 restful api 接口
